Animated!

Sobre nós

Criado a partir de uma enorme paixão por filmes de animação o Animated! é um projeto que tem como objetivo abordar técnicas e recursos da área de Computação Gráfica que são utilizados nas indústrias cinematográficas, desenvolvimento de jogos, geração de efeitos especiais...

Veja como a Computação Gráfica está presente em diversas áreas aqui.

Maya

Software de Modelagem 3D proprietário, muito presente na área de animação e desenvolvimento de jogos, já foi utilizado para animar o desenho Pokemón e auxiliou a criação de jogos famosos como Mario, Sonic the Hedgehog e Animal Crossing.
Saiba mais aqui.

Blender

Assim como o Maya, o Blender é um software de modelagem 3D, porém fornece alguns recursos a mais por ser um programa de código aberto.
Saiba mais aqui.

OpenGL

Escrito em C/C++ é uma API de código aberto, também é muito utilizadao para o desenvolvimento de jogos, ambientes 3D, aplicativos gráficos, dentre outros.
Saiba mais aqui.
